## I. Creating an Optional
```java
10: public static Optional<Double> average(int... scores) {
11:     if(scores.length == 0) return Optional.empty();
12:     int sum = 0;
13:     for(int score: scores) sum += score;
14:     return Optional.of((double) sum / scores.length);
```
